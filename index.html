SSL-HWE: Semi-supervised Learning-based Hierarchical Waypoint Extraction for Imitation Learning in Robotic Manipulation

Xinzi Liu1, Xinyu Zhai1, Junwei Liu1, Wei Zhang1
1Southern University of Science and Technology
Paper

Abstract

Imitation learning has proven effective for achieving manipulation tasks when provided with high-quality human demonstrations. However, human demonstrations often contain imperfections, injecting noise and complexity into the dataset. In this paper, we propose a novel data processing method that enhances downstream imitation learning by modifying only the action data, using a small set of labeled key intervals. Our approach consists of two main components: semi-supervised learning for key interval segmentation and optimization-based waypoint extraction. The semi-supervised learning model identifies key intervals in the unlabeled dataset, enabling full segmentation of each trajectory in the demonstrations. These interval labels are then used to solve optimization problems with interval-specific constraints to derive waypoints, which are subsequently used to adjust the actions in the demonstration dataset. Our data processing algorithm was validated through a series of real-world experiments in conjunction with a state-of-the-art imitation learning algorithm. The results demonstrate that our approach surpasses the existing data processing algorithm by 30% to 60%.


Method Overview

We propose a data processing method for human demonstration datasets, called Semi-Supervised Learning-based Hierarchical Waypoint Extraction (SSL-HWE), to enhance downstream imitation learning performance. Our approach consists of two main steps: (1) Key Interval Segmentation: We develop a semi-supervised learning model that utilizes a small labeled dataset to predict key intervals in unlabeled data. This model processes side-view image sequences to generate categorical probabilities, which are then converted into predicted labels to segment trajectories into key and non- key intervals. (2) Hierarchical Waypoint Extraction: Leverag- ing these segmented labels, we solve optimization problems with interval-specific constraints to derive waypoint selection vectors. These vectors indicate which timesteps should be selected as waypoints, and are used to adjust actions in the demonstration dataset. The pipeline of our method is illustrated in Figure 1.
